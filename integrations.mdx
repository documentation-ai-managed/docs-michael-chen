## Webhook Setup for Notifications

Set up webhooks to receive real-time events like page updates or new comments. Follow these steps to configure a webhook.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to your workspace settings and select Webhooks. Click "New Webhook".
  </Step>
  <Step title="Add Endpoint URL" icon="link">
    Enter your endpoint, such as `https://your-webhook-url.com/michaelchen`.
  </Step>
  <Step title="Select Events" icon="bell">
    Choose events like `page.updated` or `comment.created`.
  </Step>
  <Step title="Save and Test" icon="check-circle">
    Save the webhook and trigger a test event.
  </Step>
</Steps>

Here is a sample webhook payload:

<CodeGroup tabs="JSON">
```json
{
  "event": "page.updated",
  "page_id": "doc_123abc",
  "timestamp": "2024-01-15T10:30:00Z",
  "user_id": "user_456def"
}
```
</CodeGroup>

## API Integration for Custom Automation

Build custom integrations using the REST API at `https://api.example.com/v1`. Authenticate with your `{API_KEY}` in the `Authorization` header.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer `{YOUR_API_KEY}`
</ParamField>

<ParamField header="Content-Type" param-type="string" required="true">
  `application/json`
</ParamField>

<Tabs>
  <Tab title="JavaScript" icon="code">
    ```javascript
    const response = await fetch('https://api.example.com/v1/pages', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        title: 'New Doc',
        content: 'Hello from integration!'
      })
    });
    const data = await response.json();
    console.log(data);
    ```
  </Tab>
  <Tab title="Python" icon="python">
    ```python
    import requests

    headers = {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    }
    data = {
        'title': 'New Doc',
        'content': 'Hello from integration!'
    }
    response = requests.post('https://api.example.com/v1/pages', headers=headers, json=data)
    print(response.json())
    ```
  </Tab>
</Tabs>

<Response tabs="200">
```json
{
  "id": "doc_123abc",
  "title": "New Doc",
  "status": "success"
}
```
</Response>

## Embedding External Content

Embed live content from external services to keep your docs dynamic.

<Expandable title="Iframe Embedding" default-open="true">
  Use secure iframes for dashboards or tools. Example:

  ```
  <Iframe
    src="https://dashboard.example.com/embed/123"
    title="Embedded Dashboard"
    width="100%"
    height="400"
  />
  ```

  Ensure the source allows embedding via CORS.
</Expandable>

<Expandable title="Advanced Custom Scripts">
  For JavaScript embeds, add scripts cautiously:

````javascript
<script src="https://external-tool.com/widget.js" data-key="YOUR_KEY"></script>
````

  Test in preview mode first.
</Expandable>

<Callout kind="tip">
Start with webhooks for notifications, then explore API automations. Monitor integration logs in your workspace settings for troubleshooting.
</Callout>
