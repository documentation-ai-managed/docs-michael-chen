## Upgrade Instructions

Follow these steps to upgrade to the latest version.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a backup of your current documentation space.

````bash
cp -r /path/to/michael-chen /backup/location
````

  </Step>
  <Step title="Update Package" icon="download">

    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm install michael-chen@latest
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn add michael-chen@latest
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Execute database migrations if applicable.

````bash
npm run migrate
````

    Review any `{breaking changes}` from the latest release notes.

  </Step>
  <Step title="Restart Service" icon="play">

    Restart your Michael Chen instance.

````bash
systemctl restart michael-chen
# or
npm start
````

  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, verify your login form and custom themes. For issues, check the full version history or contact support.
</Callout>
